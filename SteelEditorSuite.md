# Steel Editor Suite

> **"Bespoke, high-performance tooling built to eliminate friction between asset design and engine runtime."**

**Steel Editor Suite** is DoItBetter Studio's proprietary suite of development utilities, designed specifically to build, edit, and manage content for the **Damascus Engine** and *Glyphborn*. Built from the ground up to replace generic third-party software, Steel provides a seamless, code-driven workflow for world creation, UI construction, asset management, and live scripting.

---

## Tooling Highlights

### 🗺️ World & Map Chunk Editor
* **Multi-Layered Tile Authoring:** Purpose-built for Damascus' 3D tile system, supporting seamless placement across regional, local, and interior tileset layers.
* **Massive Spatial Management:** Graphically construct and manage large matrix chunk streaming layouts with real-time previewing, vertical slope editing, and 3D collision boundaries.

### 📦 Streamlined Data & Asset Pipeline
* **Native Binary Packing:** Compiles raw world data, materials, models, and audio into highly optimized binary caches ready for instant engine consumption.
* **Zero Translation Overhead:** Eliminates heavy import/export delays by matching Damascus' cache-friendly memory structures directly.

### 🖥️ Code-First UI & System Tools
* **Lightweight Interface Design:** Rapidly author and preview game UI layouts, menus, and HUD components engineered for direct code execution without heavy markup overhead.
* **Animation & Spatial Pipelines:** Fine-tune deterministic animations, timing curves, and spatial effects with precise frame-by-frame control.

---

## Integrated Modding Workflow

Steel is natively integrated with **XenoScript**, providing modders and internal developers alike with an intuitive environment to author and validate custom game logic:

* **Live Script Debugging:** Write, compile, and test XenoScript bytecode routines directly within editor tools.
* **Content Bundling:** Package custom maps, quests, and mechanics into secure, sandboxed mod packages ready for distribution.

---

## Philosophy: Built for the Engine, Not the Market

Generic commercial editors try to be everything for everyone, resulting in bloated interfaces, sluggish iteration times, and fragile export pipelines. **Steel Editor Suite** was engineered with a single goal: give DoItBetter Studio 100% precision and speed when shaping the engine and world of *Glyphborn*.
